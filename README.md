# @youwol/geophysics


## Description
Online documentation can be found [here](https://youwol.github.io/geophysics/dist/docs/)


## Use 
To install the required dependencies:
```shell
yarn 
```
To build for development:
```shell
yarn build:dev
```
To build for production:
```shell
yarn build:prod
```

To test:
```shell
yarn test
```
